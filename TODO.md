# Push Portfolio to GitHub

Repo: https://github.com/abhi-9598/My-Portfolio.git (empty)

You are already in the correct directory. Run these commands one by one:

1. git init

2. git add .

3. git commit -m \"Initial commit: My Portfolio React app\"

4. git branch -M main

5. git remote add origin https://github.com/abhi-9598/My-Portfolio.git

6. git push -u origin main

If push prompts for credentials:
- Username: your GitHub username (abhi-9598)
- Password: GitHub Personal Access Token (PAT) - generate at https://github.com/settings/tokens with repo scope.

After push, the project will be live on GitHub.

To preview locally first: npm run dev
